# Swimming Points Calculator

Калькулятор очков World Aquatics и спортивных разрядов для пловцов. 

**Live:** [https://swimpoints.ru](https://swimpoints.ru/)

---

## Возможности

- **Мгновенный расчёт** очков World Aquatics по формуле WA
- **Определение разряда** по нормативам России
- **Поддержка бассейнов** 25м (SCM) и 50м (LCM)

### World Aquatics Points
- Базовое время взято из [официальных таблиц World Aquatics](https://www.worldaquatics.com/swimming/points)
- Период действия: 2025-2026
- Формула расчёта: P = 1000 × (Base / Time)³ (где Base - базовое время, Time - время пловца)

### Спортивные разряды России по плаванию
- [Нормативы из Единой всероссийской спортивной классификации (ЕВСК)](https://minsport.gov.ru/activity/government-regulation/evsk-1/letnie-olimpijskie-vidy-sporta-3/plavanie-3/)
- Действуют с 26.11.2024
- Включают все разряды: МСМК, МС, КМС, I-III, I-IIIю

## Использование

1. Откройте https://swimpoints.ru
2. Выберите бассейн (25м или 50м)
3. Выберите пол
4. Выберите дистанцию
5. Введите время (формат: 1:05.42 или 65.42)
7. Нажмите "Рассчитать"

## API

### API Endpoints

- `GET /events` - Возвращает список всех дистанций. Можно отфильтровать по типу бассейна: ?pool=SCM или ?pool=LCM.
- `GET /ranks` - Возвращает все нормативы для указанной дистанции и бассейна.
- `GET /calculate` - Выполняет расчет очков World Aquatics и определяет спортивный разряд по заданным параметрам (time, event, pool, gender).

### API Документация

Полная документация (Swagger UI) доступна по адресу: [https://swimpoints.ru/docs](https://swimpoints.ru/docs)
